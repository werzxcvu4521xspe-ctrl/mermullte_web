# 머물렛(Mermullet) 웹사이트 기능 명세서
> 안티그래비티(Antigravity)에 전달하는 기능 요구사항 문서입니다.
> 이 문서를 읽고 웹사이트 전체를 구현해 주세요.

---

## 서비스 개요

머물렛은 자연 감성의 독채 숙소를 소개하고, 방문자가 직접 예약과 결제까지 완료할 수 있는 숙소 예약 웹사이트입니다. 운영 숙소는 총 3개이며, 숙소 소개부터 날짜 선택, 결제 완료, 확인 이메일 수신까지 모든 과정이 이 웹사이트 안에서 이루어져야 합니다. 별도의 회원가입 없이 누구나 예약할 수 있어야 합니다.

---

## 숙소 정보

웹사이트에서 소개하고 예약받을 숙소는 아래 3개입니다.

**소나무 채** — 1박 120,000원, 최대 2인, 더블침대·욕실·미니바·WiFi 제공
**달빛 온돌방** — 1박 180,000원, 최대 2인, 온돌·욕조·테라스·미니바·WiFi 제공
**별빛 스위트** — 1박 280,000원, 최대 4인, 킹침대·욕조·샤워실·거실·주방·WiFi 제공

## 페이지 구성 및 기능 상세

### 1. 홈 페이지

방문자가 처음 접하는 페이지입니다. 머물렛의 브랜드 감성을 전달하고, 객실로 자연스럽게 이동할 수 있도록 유도합니다.

화면 상단에는 숙소의 분위기를 담은 대표 이미지와 함께 브랜드 슬로건("자연이 머무는 곳, 당신의 쉼을 위한 공간")이 표시됩니다. 그 아래에는 머물렛이 제공하는 4가지 핵심 가치(자연 속 프라이빗 공간, 감성적인 인테리어, 편안한 휴식, 따뜻한 환대)를 간단한 아이콘과 설명으로 나열합니다. 이어서 3개 숙소를 카드 형태로 미리 보여주고, 각 카드에는 사진·이름·1박 요금·간단한 설명이 포함됩니다. 카드를 클릭하면 해당 객실의 상세 페이지 또는 예약 페이지로 이동합니다. 페이지 하단에는 실제 투숙객 후기 3개를 보여줍니다.

### 2. 객실 페이지

3개 객실 각각을 상세하게 소개하는 페이지입니다.

각 객실마다 사진 여러 장을 슬라이드 형태로 볼 수 있어야 합니다. 객실 이름, 최대 인원, 1박 요금, 객실 설명 문단, 제공 어메니티 목록이 표시됩니다. 페이지 내에 "이 객실 예약하기" 버튼이 있어서 클릭 시 해당 객실이 미리 선택된 상태로 예약 페이지로 이동합니다.

### 3. 예약 페이지 — 날짜 및 정보 입력

방문자가 예약 정보를 입력하는 핵심 페이지입니다. 이 페이지는 크게 두 영역으로 나뉩니다.

**왼쪽: 날짜 선택 캘린더**

월별 캘린더가 화면에 표시됩니다. 방문자는 캘린더에서 체크인 날짜를 먼저 클릭하고, 이어서 체크아웃 날짜를 클릭해서 숙박 기간을 선택합니다. 캘린더에는 예약이 이미 완료된 날짜와 운영자가 차단한 날짜가 시각적으로 구분되어 표시되어야 하며, 이 날짜들은 클릭할 수 없어야 합니다. 오늘 이전 날짜도 선택 불가입니다. 날짜 선택 범위 안에 예약 불가 날짜가 포함되어 있으면 선택이 되지 않아야 하고, 이유를 안내하는 문구가 표시되어야 합니다. 날짜 선택이 완료되면 화면 어딘가에 "X박 / 합계 XXX,XXX원(세금 포함)" 형태로 요금이 바로 계산되어 보입니다. 요금은 1박 요금 × 숙박 일수에 10%의 세금을 더한 금액입니다.

**오른쪽: 예약자 정보 입력 폼**

날짜 선택 후 아래 정보를 입력받습니다.

객실 선택(드롭다운, 3개 중 하나), 체크인 날짜, 체크아웃 날짜, 인원 수(1~4명 드롭다운), 예약자 이름, 연락처(010-XXXX-XXXX 형식), 이메일 주소, 특이사항(선택 입력, 긴 텍스트). 모든 필수 항목이 입력되면 "다음 단계 — 결제하기" 버튼이 활성화됩니다.

**유효성 검사 규칙**

이름은 2글자 이상이어야 합니다. 연락처는 010으로 시작하는 휴대폰 번호 형식이어야 합니다. 이메일은 올바른 이메일 형식이어야 합니다. 선택한 인원이 해당 객실 최대 인원을 초과하면 경고 문구를 표시하고 다음 단계로 넘어가지 않습니다. 이미 예약된 날짜를 선택하려 하면 "해당 날짜는 예약이 불가합니다"라는 안내를 표시합니다.

### 4. 결제 페이지

예약 정보 입력 완료 후 이동하는 페이지입니다.

상단에 예약 요약이 표시됩니다. 어떤 객실을, 언제부터 언제까지, 몇 박, 몇 명, 총 얼마인지 한눈에 볼 수 있어야 합니다. 그 아래에서 결제 수단을 선택합니다. 지원하는 결제 수단은 신용/체크카드, 카카오페이, 무통장입금 세 가지입니다. 카드를 선택하면 카드 번호, 유효기간, CVC, 카드 소유자 이름을 입력하는 폼이 나타납니다. 카카오페이를 선택하면 카카오페이 결제창으로 연결됩니다. 무통장입금을 선택하면 입금 계좌 정보와 입금 기한(예약일로부터 24시간 이내)이 표시됩니다.

결제 수단 선택 및 정보 입력 후 "결제 완료하기" 버튼을 누르면 결제가 진행됩니다. 결제가 성공하면 예약 완료 페이지로 이동합니다. 결제가 실패하면 실패 이유를 안내하고 다시 시도할 수 있는 버튼을 제공합니다. 결제 정보는 SSL 암호화로 보호된다는 안내 문구가 하단에 표시됩니다.

### 5. 예약 완료 페이지

결제 성공 후 도달하는 페이지입니다.

"예약이 완료되었습니다"라는 메시지와 함께 예약 번호가 표시됩니다. 예약 번호는 MRM-날짜-랜덤숫자 형식으로 자동 생성됩니다(예: MRM-20260519-4821). 예약자가 입력한 이메일 주소로 예약 확인 이메일이 자동 발송됩니다. 이 페이지에서 홈으로 돌아가는 버튼과 예약 조회 페이지로 이동하는 버튼을 제공합니다.

### 6. 예약 조회 및 취소 페이지

회원가입 없이 예약자 본인이 예약 내역을 조회하고 취소할 수 있는 페이지입니다.

예약 번호와 예약 시 입력한 이메일 주소를 입력하면 예약 내역이 조회됩니다. 조회 결과에는 예약한 객실, 체크인·체크아웃 날짜, 숙박 일수, 인원, 결제 금액, 현재 예약 상태가 표시됩니다. 예약 상태는 "예약 확정", "취소 완료", "숙박 완료" 중 하나로 표시됩니다.

예약 상태가 "예약 확정"이고 체크인 날짜가 3일 이상 남은 경우에만 "예약 취소" 버튼이 활성화됩니다. 취소 버튼을 누르면 "정말 취소하시겠어요? 취소 시 전액 환불됩니다"라는 확인 모달이 뜨고, 확인을 누르면 취소가 처리됩니다. 취소가 완료되면 예약자 이메일로 취소 확인 이메일이 발송됩니다. 체크인 3일 미만이 남은 경우 취소 버튼이 비활성화되고 "체크인 3일 전까지만 취소 가능합니다"라는 안내가 표시됩니다.

## 이메일 자동 발송

아래 두 가지 상황에서 이메일이 자동으로 발송되어야 합니다.

**예약 확인 이메일** — 결제 완료 직후 발송됩니다. 이메일 제목은 "[머물렛] 예약이 확정되었습니다 🌿"이며, 내용에는 예약 번호, 객실명, 체크인·체크아웃 날짜 및 시간(체크인 15:00, 체크아웃 11:00), 숙박 일수, 인원, 결제 금액, 이용 안내(주차, 문의처), 취소 정책 안내가 포함됩니다.

**취소 확인 이메일** — 예약 취소 완료 직후 발송됩니다. 이메일 제목은 "[머물렛] 예약이 취소되었습니다"이며, 취소된 예약 정보와 환불 예상 일정(영업일 3~5일)이 포함됩니다.

## 관리자 페이지

숙소 운영자만 접근할 수 있는 관리 페이지입니다. 별도의 관리자 로그인(이메일+비밀번호)을 통해서만 진입할 수 있습니다. 로그인하지 않은 상태에서 관리자 주소로 접근하면 로그인 페이지로 자동으로 이동합니다.

**대시보드**

로그인 후 첫 화면입니다. 이번 달 예약 건수, 이번 달 총 매출, 이번 달 총 지출, 이번 달 순이익, 오늘 체크인 예정 건수, 오늘 체크아웃 예정 건수를 숫자로 보여줍니다. 각 수치는 페이지를 새로 고침하지 않아도 실시간으로 갱신됩니다. 그 아래에 최근 예약 5건을 간략하게 보여주는 목록이 있고, 이번 달 매출과 지출을 한 화면에서 비교하는 간단한 막대 그래프가 함께 표시됩니다. 재무 관리 화면으로 바로 이동하는 버튼도 대시보드에 노출됩니다.

**예약 목록**

전체 예약 내역을 표 형태로 볼 수 있습니다. 한 페이지에 20건씩 표시되며, 페이지를 넘길 수 있습니다. 객실별, 예약 상태별, 날짜 범위로 필터링할 수 있습니다. 예약자 이름, 연락처, 예약 번호로 검색할 수도 있습니다. 각 예약 행을 클릭하면 상세 정보를 볼 수 있고, 운영자가 예약 상태를 직접 변경(확정·취소·완료 처리)할 수 있습니다.

**차단 날짜 관리**

객실별로 특정 날짜를 예약 불가 상태로 설정하거나 해제할 수 있습니다. 월별 캘린더 형태로 표시되며, 날짜를 클릭해서 차단 여부를 토글합니다. 차단할 때 메모(예: "수리", "개인 일정")를 남길 수 있습니다. 이미 예약이 잡혀 있는 날짜는 별도 색으로 표시되어 구분됩니다.

**객실 정보 관리**

각 객실의 1박 요금, 설명 문구, 제공 어메니티 목록을 수정할 수 있습니다. 객실을 일시적으로 비활성화하면 예약 페이지에서 해당 객실이 선택 불가 상태가 됩니다.

**재무 관리**

운영자가 매출, 지출, 순이익을 한 곳에서 실시간으로 확인할 수 있는 재무 관리 화면입니다. 이 화면은 크게 세 영역으로 구성됩니다.

첫 번째 영역은 재무 요약 대시보드입니다. 화면 상단에 이번 달 기준으로 총 매출, 총 지출, 순이익 세 가지 숫자를 나란히 크게 표시합니다. 순이익은 총 매출에서 총 지출을 뺀 값이며 자동으로 계산됩니다. 숫자 아래에는 지난달 대비 증감을 퍼센트로 함께 보여줍니다. 예를 들어 지난달보다 매출이 늘었으면 초록색으로 "+12%", 줄었으면 빨간색으로 "-8%" 형태로 표시됩니다. 조회 기간을 이번 달, 지난달, 올해 전체, 직접 날짜 지정 중에서 선택할 수 있으며 선택 즉시 모든 수치가 해당 기간 기준으로 다시 계산되어 표시됩니다.

두 번째 영역은 매출 현황입니다. 매출은 예약 결제가 완료된 금액을 기준으로 자동 집계됩니다. 별도로 입력하지 않아도 예약 시스템과 연동되어 결제 완료 건은 즉시 매출에 반영됩니다. 취소 환불이 발생한 경우 해당 금액은 매출에서 자동으로 차감됩니다. 매출 현황 표에는 날짜, 예약 번호, 객실명, 결제 금액, 세금, 실수령액이 항목별로 나열됩니다. 실수령액은 결제 금액에서 세금(10%)과 결제 수수료(카드 결제 기준 2.5%)를 제외한 금액입니다. 객실별 매출 비중을 도넛 차트로 시각화하여 어떤 객실에서 매출이 가장 많이 발생하는지 한눈에 파악할 수 있습니다. 월별 매출 추이를 막대 그래프로 보여주며, 최근 6개월 또는 12개월 기준으로 전환할 수 있습니다.

세 번째 영역은 지출 관리입니다. 운영자가 직접 지출 내역을 입력하고 관리할 수 있습니다. 지출 항목을 추가할 때는 날짜, 카테고리, 항목명, 금액, 메모(선택)를 입력합니다. 카테고리는 청소비, 소모품, 수도·전기·가스, 수리·유지비, 식음료(어메니티), 마케팅·광고, 플랫폼 수수료, 기타 중에서 선택합니다. 카테고리는 운영자가 직접 추가하거나 삭제할 수 있습니다. 입력된 지출 내역은 표 형태로 나열되며, 날짜 기준으로 정렬됩니다. 각 항목의 수정과 삭제가 가능합니다. 카테고리별 지출 합계를 막대 그래프로 시각화하여 어떤 항목에 지출이 집중되는지 파악할 수 있습니다.

데이터 내보내기 기능도 제공합니다. 조회 중인 기간의 매출 내역, 지출 내역, 순이익 요약을 엑셀 파일(.xlsx)로 내려받을 수 있습니다. 내려받은 파일에는 매출 시트와 지출 시트가 분리되어 있으며, 요약 시트에 기간 합계가 정리되어 있습니다.

## 예약 불가 처리 규칙

아래 두 가지 경우 해당 날짜는 예약이 불가능합니다.

첫째, 해당 날짜에 이미 확정된 예약이 존재하는 경우입니다. 체크인 날짜부터 체크아웃 전날까지의 모든 날짜가 예약 불가로 처리됩니다. 예를 들어 6월 1일 체크인, 6월 3일 체크아웃이면 6월 1일과 6월 2일이 예약 불가입니다.

둘째, 관리자가 차단 날짜로 등록한 경우입니다.

두 조건을 모두 실시간으로 반영해서 캘린더에 표시해야 합니다. 방문자 두 명이 동시에 같은 날짜를 예약하려 할 때 한 명만 성공하고 나머지는 "이미 예약된 날짜입니다"라는 안내를 받아야 합니다.

## 금액 계산 기준

모든 요금 계산은 아래 방식을 따릅니다.

객실 1박 요금 × 숙박 일수 = 객실 요금 합계
객실 요금 합계 × 10% = 세금 및 봉사료
객실 요금 합계 + 세금 및 봉사료 = 최종 결제 금액

최종 금액은 원 단위로 반올림합니다. 예약 당시의 1박 요금을 기준으로 계산하며, 이후 운영자가 요금을 변경해도 기존 예약 금액은 바뀌지 않습니다.

## 전체 예약 흐름 요약

방문자가 웹사이트에 접속합니다 → 홈 또는 객실 페이지에서 숙소를 둘러봅니다 → 예약 페이지에서 객실과 날짜를 선택합니다 → 예약자 정보를 입력합니다 → 결제 페이지에서 결제 수단을 선택하고 결제를 완료합니다 → 예약 완료 페이지에서 예약 번호를 확인합니다 → 예약 확인 이메일을 수신합니다 → 필요 시 예약 조회 페이지에서 내역을 확인하거나 취소합니다.
