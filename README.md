# mermullet_web 🏨

호텔 소개 페이지 및 실시간 예약 기능을 제공하는 웹 어플리케이션 프로젝트입니다.

## 🚀 프로젝트 로드맵

1. **GitHub 연동 및 초기 설정 (진행 중)** 🛠️
   - 로컬 Git 리포지토리 설정
   - GitHub 원격 리포지토리 연동
2. **클라우드 서버 및 데이터베이스(DB) 구축** ☁️
   - AWS 또는 기타 클라우드 서비스 환경 설정
   - DB 인프라 설계 및 연동
3. **호텔 소개 페이지 개발** 🌐
   - 세련되고 모던한 UI/UX 구현 (Glassmorphism, 다크 모드, 애니메이션 등)
   - 객실 안내 및 부대시설 소개
4. **실시간 예약 및 관리 기능 구현** 📅
   - 객실 예약 캘린더
   - 실시간 예약 정보 처리 및 DB 저장
   - 예약 확인/취소 기능

## 🛠️ 기술 스택 (예정)
- **Frontend**: Vite / Next.js, Vanilla CSS, JS
- **Backend**: Node.js (Express) or Next.js API Routes
- **Database**: PostgreSQL / MongoDB / MySQL (선택 예정)
- **Infrastructure**: Vercel / AWS / Docker (선택 예정)
